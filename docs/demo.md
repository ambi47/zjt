# 演示与提交建议

## 本地启动

```bash
npm install
npm run dev
```

访问：
- 前台：`http://localhost:3000`
- 后台：`http://localhost:3000/admin.html`

## 后台账号

- 用户名：`admin`
- 密码：`admin123456`

如需修改后台密码，可在启动前设置环境变量：
- `ADMIN_PASSWORD`
